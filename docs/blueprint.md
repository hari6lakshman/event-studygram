# **App Name**: Events-Studygram

## Core Features:

- Event Listing: Display a list of upcoming hackathons and events.
- Event Details: Provide detailed information about each event, including descriptions, dates, and registration links.
- Organizer Login: Allow organizers to log in to upload and manage events.
- Organizer Sign-up: Enable new organizers to sign up and create accounts.
- Event Upload: Allow logged-in organizers to upload new event information, with fields for event name, description, date, and registration URL.

## Style Guidelines:

- Background: Deep, rich blue (#1746A3) for a sophisticated feel.
- Card Background: Slightly darker blue (#0F3075) for content separation.
- Primary/Accent Color: Vibrant gold (#F0AB23) for borders and interactive elements.
- Body Text/Foreground: Soft, off-white (#D9E2EC) for readability.
- Headline Font: 'Playfair', serif, for main headlines. Note: currently only Google Fonts are supported.
- Body Font: 'PT Sans', sans-serif, for body text and chat messages. Note: currently only Google Fonts are supported.
- Central, card-like container with large rounded corners and a distinct gold border to stand out against the background.
- Header and footer separators feature stylish horizontal gradients that fade from transparent to the gold accent color in the middle.
